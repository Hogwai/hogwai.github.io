## Introduction

Fonctionnalités disponibles pour générer des projets Micronaut via la CLI.

Elles peuvent être consultées grâce au flag `--list-features` de la commande de création utilisée :

```shell
mn create-app --list-features
```

Un script pour faciliter la recherche est disponible sur le [dépôt GitHub find-feature.sh](https://github.com/Hogwai/hogwai.github.io-content/blob/main/micronaut/cli/find-feature.sh).
